# 2026-06-14 — Início do vault Obsidian do EvoSync

> Daily note âncora. Marca o dia em que o vault `EvoSyncOB/` foi estruturado
> como knowledge base oficial do projeto.

## O que aconteceu

Decidiu-se integrar o projeto EvoSync ao Obsidian para ter uma base de
conhecimento versionada, navegável e profissional.

## Decisões do dia

- **Opção A — Espelho simples** (sem plugins da comunidade; só core plugins já ativos).
- **Vault dentro do repo** (`EvoSyncOB/`, versionado via git junto com o código).
- **Estrutura completa** com 3 MOCs, 3 ADRs, 5 docs importadas, 1 template, 1 daily note.

## ➕ Update 2026-06-14 (parte 2) — Documentação do modelo SaaS

Decidiu-se que o EvoSync é vendido como **SaaS hospedado** (você é o provedor,
cliente paga mensalidade fixa). Foram criados:

- [[ADR-004-Modelo-SaaS-Hospedado]] — decisão: você hospeda TUDO, 1 Evolution API por tenant
- [[Runbook-Onboarding-Cliente]] — como provisionar um cliente novo (~15-20 min)
- [[Guia-Cliente-Primeiros-Passos]] — manual do cliente B2B (linguagem leiga)
- [[Runbook-Cobranca-Licenca]] — ciclo de mensalidade, renovar, suspender
- [[Runbook-Suporte-Diagnostico]] — playbook de troubleshooting
- [[ADR-002-Evolution-API-BYOTenant]] — atualizado: BYO vira exceção avançada

Agora o vault cobre o ciclo completo: **arquitetura → operação → onboarding → cobrança → suporte**.

## ➕ Update 2026-06-14 (parte 3) — Redesign do Painel Administrativo

O admin estava visualmente quebrado (login dark, painel light, componentes
genéricos). Redesenhado do zero com base em Linear + Vercel.

- [[Design-System-v2]] — visão geral do redesign
- [[Design-Tokens]] — cores HSL, tipografia, espaçamento, sombras, animações
- [[Componentes]] — catálogo dos novos componentes
- [[Direcao-Visual]] — referências, anti-referências, decisões

Mudanças no código:
- `tailwind.config.ts` reescrito (paleta semântica completa, animações, fontes)
- `globals.css` reescrito (CSS variables para light/dark, base layer)
- Novos: StatCard, PageHeader, StatusBadge, EmptyState, ConfirmDialog, CommandPalette, ThemeToggle
- Redesenhados: AdminShell, login split-screen, dashboard, todas as tabelas admin
- App do operador também alinhado (sidebar + header) para coerência
